## renren-bak
人人网资料备份python脚本 write by rublog

通过运行本脚本来保存人人网上的说说、博客文章、相册照片

仅此而已。

运行需要安装的软件：

*Python 3.5 +（理论上3.3之后的版本应该都行，我调试的版本是windows 8.1 64位，python 3.5 32位）

*依赖：pypi requests
